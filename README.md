# PI Project Godot

## Overview
This project is a game developed using the Godot game engine. The goal of the project is to create an engaging and interactive experience.

## Project Structure
- `scenes/`: Contains all the scene files for the game.
- `scripts/`: Contains GDScript files for different functionalities.
- `assets/`: Contains graphical and audio assets used in the game.
- `project.godot`: The main configuration file for the Godot project.

## Setup Instructions
To set up the project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/joaopgamarques/PI-Project-Godot.git
   cd PI-Project-Godot
   ```
2. Open the project in Godot:
   - Open Godot and select **Import**.
   - Navigate to the directory where the project is cloned and select `project.godot`.

## Running the Project
To run the project:
1. Open the project in Godot.
2. Press F5 or click on the **Play** button to start the game.
